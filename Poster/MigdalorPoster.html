הצורך והמטרה

כיום, התקשורת בקהילת "מגדלי הים התיכון" מפוזרת בין פלטפורמות רבות (קבוצות וואטסאפ, לוחות מודעות), מה שמקשה על העברת מסרים, הרשמה לאירועים, ויוצר קשיי השתלבות לדיירים חדשים.


מטרת הפרויקט היא ליצור פלטפורמה דיגיטלית אחת, "מגדל-אור", שתאחד את כלל השירותים והמידע. האפליקציה שואפת לשפר את התקשורת, לחזק את הקהילה, ולהוות מקור מידע מרכזי ונוח לשימוש עבור הדיירים וההנהלה כאחד.

מאפייני המערכת

  • פרופילים וספר טלפונים: יצירת פרופיל אישי וגישה נוחה לפרטי קשר של כלל הדיירים.
  • לוח מודעות ויוזמות: פרסום והרשמה לחוגים, אירועים ופעילויות קהילתיות.
  • מרקטפלייס קהילתי: זירת מסחר פנימית למכירה, קנייה ומסירה של פריטים.
  • נוהל "בוקר טוב": מנגנון יומי פשוט לבדיקת שלומם של הדיירים.
  • כלים ניהוליים: ממשק ייעודי לוועד המאפשר פרסום הודעות, ניהול פעילויות והפקת דוחות.

מבנה הפרויקט

  • צד לקוח (Frontend): אפליקציית מובייל שפותחה באמצעות React Native ו-Expo, המאפשרת תאימות למערכות הפעלה iOS ו-Android.
  • צד שרת (Backend): שרת API שנבנה על גבי ASP.NET Core, האחראי על הלוגיקה העסקית, ניהול המידע והתקשורת עם בסיס הנתונים.
  • בסיס נתונים (Database): מערכת MS SQL Server לאחסון מאובטח ומהימן של כלל נתוני המערכת.

אלמנט חכם: סוכן AI

כדי להפוך את האפליקציה לקלה ואינטואיטיבית ככל האפשר, שילבנו סוכן חכם מבוסס בינה מלאכותית.

הסוכן מאפשר למשתמשים לתקשר עם האפליקציה בשפה טבעית. לדוגמה, דייר יכול לשאול "מתי מתקיים חוג היוגה?" או "הצג לי את פרטי הקשר של משה כהן", והמערכת תציג לו את המידע הרלוונטי באופן מיידי.
